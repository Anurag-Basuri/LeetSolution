# 🤝 Disjoint Set Union (DSU)

This directory contains problems solved using the Disjoint Set (Union-Find) data structure.

## Overview
Disjoint Set Union (DSU) is highly effective for:
- Finding connected components in a graph
- Cycle detection
- Kruskal's Minimum Spanning Tree algorithm
- Dynamic connectivity queries

> Note: Many graph problems can be solved using both DFS/BFS and DSU. When a problem has multiple valid approaches, we categorize it by the primary or most optimal approach used.
